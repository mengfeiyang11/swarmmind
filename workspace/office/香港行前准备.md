# 香港行前准备清单

## 一、 证件与过关准备
1. **港澳通行证 (EEP)**：确保有效期在6个月以上。
2. **有效签注**：检查是否有有效的香港签注（旅游签注通常为G签或L签，L签现在也可自助过关，无需找旅行社送关）。
3. **身份证**：乘坐交通工具或办理酒店入住时可能需要出示。
4. **过关小票**：过关时海关打印的入境小白条，在香港期间请务必妥善保管，离境时可能会检查。

## 二、 货币与支付
1. **港币现金**：准备适量现金，主要用于部分传统茶餐厅、街边小吃、小巴士或购买八达通卡。（建议提前在内陆银行兑换，汇率较好）。
2. **八达通卡**：强烈建议办理一张实体八达通（或在手机开通电子八达通），可用于地铁、巴士、便利店、自动售卖机及部分餐厅消费。
3. **移动支付**：微信和支付宝在香港绝大多数商场、连锁店（如万宁、屈臣氏）及部分餐厅通用，且经常有汇率优惠。
4. **信用卡**：带有Visa、MasterCard或银联标志的信用卡，方便大额消费。

## 三、 网络与通讯
1. **手机漫游 / 流量包**：提前在支付宝/运营商APP购买境外流量包，或开通国际漫游。
2. **香港电话卡 / eSIM**：如需大量上网或需要香港本地号码（用于拨打本地电话、点外卖、收验证码），可提前在淘宝购买上网卡。

## 四、 必备APP推荐
1. **Google Maps (谷歌地图)**：香港最精准的导航软件，可查看公交、地铁、步行路线及商铺营业时间。
2. **MTR Mobile**：港铁官方APP，查询地铁线路和首末班车时间。
3. **OpenRice (开饭喇)**：香港本地最流行的美食点评APP，相当于大陆的大众点评。
4. **KMB / 新巴城巴**：查询公交车实时到站信息的APP。
5. **Uber**：在香港打车较为方便，且价格透明。
6. **QuickFX**：实时汇率查看APP，方便计算消费价格。

## 五、 生活与日用品
1. **转换插头**：香港使用的是**英标三孔插头**，务必携带转换器（部分酒店提供USB接口，但自带最稳妥）。
2. **充电宝**：出门在外导航、拍照耗电快，需随身携带大容量充电宝。
3. **薄外套/雨伞**：香港商场、地铁冷气非常足，建议随身带一件薄外套；天气多变，建议备好晴雨伞。
4. **个人药品**：常备的肠胃药、感冒药、创可贴等（香港买部分处方药需医生证明，建议自带常用药）。

## 六、 重要注意事项
1. **语言沟通**：绝大多数商场、餐厅的服务人员都会说普通话或能听懂普通话，沟通基本无障碍，菜单也多为中英双语。
2. **公共交通礼仪**：乘坐扶手电梯时，请务必**靠右站立**，左侧留给急行的人。地铁内严禁饮食。
3. **禁烟规定**：香港对吸烟管控极其严格，凡是有屋顶的地方几乎全都禁烟，违规吸烟罚款非常重（最高可罚款5000港币）。请寻找指定的垃圾桶吸烟区。
4. **打车注意**：香港出租车按颜色区分运营区域（红色为市区，绿色为新界，蓝色为大屿山），打车需在指定的士站招手，上车注意系好安全带（前后排都必须系）。
